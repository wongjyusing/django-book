## 什么是Django？  
Django是python的一个Web框架。  
好了，说完了。  
与其说Django是什么，不如把问题转化为**Django能做什么？**  
Django可以做网站、api接口、管理后台等项目。  
学习Django不如先做一个**个人博客**项目来边做边学习。  
## 需要什么知识？  
* Python：变量、函数、包引用、类继承、for循环、列表、字典。  
* 三剑客：html、css、javascript。  
* Markdown语法（写博文用的）  
* 一点点网络知识  

## 什么是博客？  
像[porksuimai.com](www.porksuimai.com)、[陳鍾誠](https://mdbookspace.com/view/ccc/README.md)、[杨仕航的博客](http://yshblog.com/)、[TendTocode](http://www.tendcode.com/)都是博客，其实说成是**博客**，不如说成是**笔记本**吧。  
其实有点像在写日记本之类的网站。不过我们都是写一些关于技术方面的文章。
